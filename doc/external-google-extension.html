Google Extension Install Instructions

Download

  1. Download the Zip file from the link
  2. Unzip the file
  3. Get the Extention folder (contains a content file and a manifest file)

Install

  1. Click the three dots at the upper right corner of Chrome
  2. Go to More tools > Extensions
  3. Enable the Developer mode at the upper right corner of the page
  4. Click Load Unpacked and select the Extension folder
