User Details




  • ID
    The user's account id
  • Last active
    The last time user's client was READY
  • Nickname
    The user's server nickname
  • Badges
    The user's account badge
  • Timeout?
    Check if the user is timed out and when will the timeout end
  • Boosted?
    Check if the user boosted the server and when
  • Bot?
    Check if the user is a bot
  • Team?
    Check if the user is a team
  • Highest role
    The user's higest role they have in the server
  • Member List role
    The role that separates the user in the member list
  • Boost role
    The role the user gets if they boosted the server
  • Bot role
    The role the user gets if they are a bot (Different for every bot)
  • Color role
    The role that gives the user the color they are currently having
  • Icon role
    The role that gives the user the icon they are currently having
  • All roles
    A list of the user's roles (Highest to lowest)
  • Create date
    The timestamp when the user created their account
  • Join date
    The timestamp when the user joined the server
  • Avatar URL
    The user's avatar URL
  • Server Avatar URL
    The user's server avatar URL
